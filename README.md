# Todo Calendar 📅

A full-stack Todo Calendar web app built with Python Flask, SQLite and Bootstrap.

## 🌐 Live Demo
Coming soon...

## ✨ Features
- User Registration & Login
- Session-based authentication
- Personal calendar for each user
- Add tasks to any date (past, present, future)
- Navigate any month and year
- Mark tasks as done
- Delete tasks
- Task completion counter
- Today highlighted in yellow
- Dates with todos highlighted in blue

## 🛠 Tech Stack
- Python + Flask
- SQLite Database
- Bootstrap 5
- Jinja2 Templates
- Flask Sessions

## 📁 Project Structure
```
TodoCalendar/
├── app.py
├── database.py
└── templates/
    ├── index.html
    ├── day.html
    ├── login.html
    └── register.html
```
## ▶ How to Run
1. Install Flask: `pip install flask`
2. Run database setup: `python database.py`
3. Start server: `python app.py`
4. Visit: `http://127.0.0.1:5000`
